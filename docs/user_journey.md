# The VRYPTIC User Journey

1. **Capture:** The user takes a photo with the **VRYPTIC Camera**. The NPCC algorithm creates the first Truth Seal.
2. **Attest:** The **Solana Mobile Stack** signs the seal; the record is pushed to the Solana Ledger.
3. **Enhance:** The user edits the photo in the **VRYPTIC Editor**. Every change is logged as a provenance-safe event.
4. **Permanentize:** The final asset and its edit history are anchored via **Arweave Sentinels**.
5. **Verify:** Third parties use the VRYPTIC Verifier to confirm the asset’s origin and history.
